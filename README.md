# 📊 Exploratory Data Analysis (EDA) & Data Cleaning Portfolio

## 🚀 Overview
This repository contains a collection of Data Science projects focused on data wrangling, cleaning, and Exploratory Data Analysis (EDA). The primary goal of these projects is to take raw, messy datasets and transform them into clean, actionable insights using Python.

## 📂 Projects Included

### 1. FIFA Dataset Cleaning (`Fifa Data Cleaning.ipynb`)
* **Objective:** To clean and standardize a highly unstructured real-world dataset of FIFA players.
* **Techniques Used:** Handling missing values (NaNs), data type conversions, complex string manipulations, and standardizing column formats to prepare the data for downstream Machine Learning tasks.

### 2. Netflix Content Trend Analysis (`Netflix Solved Lab.ipynb`)
* **Objective:** To perform EDA on Netflix's global catalog to uncover content addition trends and library distribution.
* **Techniques Used:** Time-series feature extraction (parsing date objects), categorical data grouping, and visualizing content distribution (Movies vs. TV Shows) to identify business trends.

### 3. Bikeshare Data Analysis (`Bikeshare Analysis.html`)
* **Objective:** To analyze user demographics and ride statistics for an urban bikeshare system.
* **Techniques Used:** Aggregating usage statistics, computing descriptive metrics, and generating static HTML reports for stakeholder viewing.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Data Manipulation:** `pandas`, `numpy`
* **Data Visualization:** `matplotlib`
* **Environment:** Jupyter Notebook

## ⚙️ How to View
Simply click on any of the `.ipynb` or `.html` files above to view the code, cell outputs, and data visualizations directly within the GitHub web interface. No local installation is required.
